# Iconic Exam Maker

Create professional exam papers by cropping questions from PDFs and exporting clean, branded papers.

## Quick Start
1. Open the app.
2. Create a new project.
3. Import PDFs and crop questions.
4. Arrange questions in the Builder.
5. Export the paper (PDF/DOCX).

## Fonts
For best Tamil/Sinhala rendering, place the following fonts in `assets/fonts`:
- `times.ttf`, `timesbd.ttf`, `timesi.ttf`, `timesbi.ttf`
- `latha.ttf`
- `iskpota.ttf`

## Troubleshooting
- If PDFs fail to open, check the file and try again.
- If text appears incorrect, make sure the required fonts are installed/bundled.
